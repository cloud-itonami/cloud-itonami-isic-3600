# Operator Guide

## First Deployment

1. Register assets, sample points and responsible operators.
2. Import historical lab and sensor data.
3. Run read-only threshold checks.
4. Configure alert recipients and escalation paths.
5. Publish a dry-run public report.

## Minimum Production Controls

- calibrated sensor provenance
- lab data source history
- emergency alert escalation
- audit export for every public claim
- backup manual notification process

## Certification

Certified operators must prove alert integrity, evidence-backed reporting and
human review for emergency-affecting actions.
